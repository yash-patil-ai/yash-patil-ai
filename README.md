## 🚀 Featured Project

<div align="center">

### 🌾 CropSense AI — Crop Yield Prediction

> **ML-powered crop recommendation system** built with SVM algorithm  
> Predicts the best crop based on soil (N, P, K) and climate (humidity, rainfall) data

[![Streamlit](https://img.shields.io/badge/Live%20Demo-Coming%20Soon-27ae60?style=for-the-badge&logo=streamlit&logoColor=white)](#)
[![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)](#)
[![SVM](https://img.shields.io/badge/Algorithm-SVM-orange?style=for-the-badge)](#)

| Feature | Detail |
|--------|--------|
| 🤖 Algorithm | Support Vector Machine (SVM) |
| 📊 Input | N, P, K soil values + Humidity + Rainfall |
| 🌿 Output | Best crop recommendation + Top 3 with confidence |
| 🎨 UI | Dark-green themed Streamlit dashboard |
| 📍 Dataset | Indian Crop Dataset |
